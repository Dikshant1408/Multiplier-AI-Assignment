========================================
HOW TO RUN THIS PROJECT - SIMPLE GUIDE
========================================

STEP 1: PREPARE THE DATA
-------------------------
Open terminal/command prompt and run:

    python clean_data.py
    python analyze.py

You should see messages saying data was cleaned and analyzed.


STEP 2: START THE BACKEND
--------------------------
Open a NEW terminal and run:

    cd backend
    pip install -r requirements.txt
    uvicorn main:app --reload

Keep this terminal OPEN! You should see:
"Uvicorn running on http://127.0.0.1:8000"


STEP 3: START THE FRONTEND
---------------------------
Open ANOTHER NEW terminal and run:

    cd frontend
    npm install
    npm start

Your browser will automatically open to http://localhost:3000


THAT'S IT! 🎉
-------------
You should now see the dashboard with:
- Revenue chart
- Customer table (click headers to sort!)
- Category charts
- Regional cards


TROUBLESHOOTING
---------------
Problem: "Module not found" error
Solution: Run "pip install pandas numpy fastapi uvicorn"

Problem: Frontend won't start
Solution: Make sure Node.js is installed, then run "npm install" again

Problem: No data showing
Solution: Make sure you ran Step 1 first!

Problem: Port already in use
Solution: Close other programs using port 8000 or 3000


WHAT EACH FILE DOES
-------------------
clean_data.py       - Cleans messy data
analyze.py          - Analyzes data and creates reports
backend/main.py     - API that serves data
frontend/src/App.js - Dashboard website
data/raw/           - Original data files
data/processed/     - Cleaned data files
